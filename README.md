「迷失」象徵主角瑪朵在現實中失去了自我，失去了生活的方向與希望。
「Missing」除了意指失蹤，也代表她在糖果世界中遺失了記憶與存在的真實意義。

我們遊戲主要玩法是2D捲軸探索敘事型解謎遊戲，玩家將扮演主角瑪朵蒐集記憶碎片解開謎題，整體會有一張集點卡作為主要故事推動的物件，每收集一個記憶碎片就會蓋上一點，記憶碎片會顯示主角在現實中的記憶，並獲得新的技能或武器。
最後收集完所有的記憶，玩家將面臨雙重結局的抉擇，是否留在糖果王國沉睡與虛假的甜蜜，又或者勇敢面對現實的殘酷。

世界觀的設定是在述說，糖果甜甜的總是會讓人一口接著一口，儘管背後有著許多對人體的危害，但人們卻沉溺於甜蜜感當中，因此這也是我們選擇糖果做為世界觀的原因。

故事綱要
主角是一位熱愛甜點的甜點師，因為生活的壓力讓他喘不過氣，她選擇了自盡，雖然在朋友的即時發現保住了一命，但她依舊昏迷，意識也進入了糖果世界，在這個世界她忘記自己是誰，這裡的居民也是沒有煩惱天天玩樂，當主角沉溺於這裡的時候腦海出現了一個聲音，推動主角往前探險找回自我。

玩法主要是2D卷軸遊戲，過程中會有打鬥和解謎的元素。
打怪會掉落奶油能量，奶油能量達到一個數值可以恢復血量或發射大招
記憶碎片會讓玩家獲得新技能二段跳、衝刺，最後主角想起自己是誰後會有內心的自己作為最後的BOSS阻擋玩家離開。
